# WinningWoofs
A Spring Boot API and DAO SQL solution for connecting people with available puppies, featuring RestClient integration for real-time external data.

Key Features:

Java Spring Boot Backend: Provides a secure and scalable RESTful API foundation.

Data Persistence: Manages puppy availability data using a DAO SQL relational database.

MVC Architecture: Implements a clear separation of concerns with a dedicated MVC controller.

External API Integration: Utilizes a RestClient to consume and process data from outside APIs, ensuring real-time puppy information.
